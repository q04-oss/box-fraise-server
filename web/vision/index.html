Vision

Box Fraise is building an identity network rooted in showing up.

Most sign-in systems accept anyone with an email address. That's cheap to fake, and it leaves every company using them treating each user as unknown by default. We think identity works better when it's earned in person — with a signature from a device only you hold, and a private conversation with a trained Box Fraise consultant who signs off with their own reputation attached.

The consultation is unhurried. The consultant asks the questions; the user doesn't fill in a form. No government ID is checked or stored — we're building our own credential, not verifying somebody else's. What's kept about a user is minimal and human. What the user walks out with is a physical card, the artifact of that specific conversation.

Once a user is verified with us, that status becomes something other products can trust. We open it as a login API — one in-person moment, wider trust downstream. (A minimal version of that API exists today at /v1/oauth, deliberately labelled unsupported: the project is too small right now to properly maintain an integration surface. It'll harden when a real partner is waiting for it.)

We're seeding the community through hair. Hair events, hair salons, eventually the software that runs them. Hair is a licensed trade with real in-person craft; stylists are already professionals at private consultation. Same discipline the identity system depends on.

Our events are socialising first — a boy band audition, an opening night, something worth showing up for on its own terms. The consultations happen alongside, in a quieter room, at the pace of a real conversation. What draws people in is the front of the room. What turns them into verified members is the back.

The server is live. The app is written. The first event is on the calendar.

Vision

Box Fraise is building an identity network rooted in showing up.

Most sign-in systems accept anyone with an email address. That's cheap to fake, and it leaves every company using them treating each user as unknown by default. We think identity works better when it's earned in person — at a real event, in front of a real organiser, with a signature from a device only you hold.

Once a user is verified with us, that status becomes something other products can trust. We open it as a login API — one in-person moment, wider trust downstream.

We're seeding the community through hair. Hair events, hair salons, and eventually the software that runs them. Hair is a licensed trade with real in-person craft — the same discipline the identity system depends on. It's also how the project intends to fund itself; more on that in due course.

The server is live. The app is written. First events are being planned.
